# Web3.js

[![web3js](https://ethereum.consensys.net/hs-fs/hubfs/web3js.png?width=500&name=web3js.png)](http://bit.ly/web3js-portal)

**Web3.js** 是以太坊JavaScript API，能连接通用JSON RPC规范。只有运行本地或远程以太坊节点时才能使用此库。
