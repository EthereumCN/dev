# Ropsten

[![Ropsten](https://ethereum.consensys.net/hs-fs/hubfs/Ropsten.png?width=400&name=Ropsten.png)](http://bit.ly/ropsten-portal)

**Ropsten**是采用工作量证明共识机制的以太坊测试网，最近似于以太坊主网。在此测试网上，用户可轻松体验开采以太坊测试币。
