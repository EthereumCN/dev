# Slither

[![&#x56FE;&#x7247;alt](../../.gitbook/assets/logo.png)](https://github.com/crytic/slither)

Slither是一种用Python3编写、针对Solidity的数据安全性分析框架，能够快速检测漏洞，显示智能合约细节信息，提供API供开发者编写自定义脚本来审计Solidity代码。
